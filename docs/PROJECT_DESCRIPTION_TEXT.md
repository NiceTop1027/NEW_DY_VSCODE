## ⚡️ 1. 한 줄 소개 (슬로건 & 소제목)
*   **슬로건**: "설치 없이 URL 클릭 한 번으로 시작하는 우리 학교만의 클라우드 코딩 실습실, DY Web IDE"
*   **핵심 가치**: "Anywhere, Anytime, No Setup. 브라우저가 곧 컴퓨터가 되는 경험."

## 📝 2. 핵심 요약 (Abstract) - 보고서 개요/오프닝용
**DY Web IDE**는 기존 코딩 교육 현장에서 발생하는 복잡한 개발 환경 설정 문제를 해결하기 위해 개발된 **웹 기반 통합 개발 환경(IDE)**입니다. 별도의 프로그램 설치 없이 웹 브라우저만으로 C, Python, JavaScript 등 다양한 언어를 즉시 실행할 수 있으며, 실제 현업에서 사용하는 **VS Code의 UI/UX**를 완벽하게 구현하여 학생들에게 실무와 동일한 경험을 제공합니다. 또한 **Docker 컨테이너 기술**을 통해 사용자별로 안전하게 격리된 실행 환경을 보장하고, **GitHub 연동**을 통해 과제 제출과 포트폴리오 관리를 자동화한 혁신적인 교육 플랫폼입니다.

## 📖 3. 상세 소개 (Full Description) - 발표 대본/보고서 본문용

### [문제 의식: 교실에서의 혼란]
코딩 수업이 있는 날이면 컴퓨터실은 늘 소란스럽습니다. 컴퓨터가 초기화되어 파이썬을 다시 설치해야 하거나, 학생마다 집에서 사용하는 운영체제가 달라 예제 코드가 실행되지 않는 등 **'환경 설정'**에만 수업 시간의 절반을 허비하는 일이 빈번합니다. 아이디어를 코드로 구현하는 법을 배우기도 전에, 복잡한 설치 과정에 지쳐 흥미를 잃는 학생들을 보며 이 프로젝트를 기획하게 되었습니다.

### [해결책: 접속이 곧 준비의 끝]
**DY Web IDE**는 이러한 진입 장벽을 기술로 허물었습니다. 사용자는 무거운 프로그램을 설치할 필요 없이 웹 브라우저(Chrome, Edge)로 접속만 하면 됩니다. 5초 이내에 로딩되는 친숙한 **VS Code 스타일의 에디터**에서 바로 코드를 작성하고, `실행` 버튼 하나로 즉시 결과를 확인할 수 있습니다.

### [기술적 차별성: 웹 그 이상의 성능]
단순히 웹에서 텍스트만 편집하는 것이 아닙니다. 저희는 **WebAssembly**와 **Docker** 기술을 하이브리드로 적용했습니다. 가벼운 연산은 브라우저 자체에서 처리하여 빠른 속도를 보장하고, 보안이 중요한 시스템 명령어나 무거운 컴파일 작업은 서버의 격리된 Docker 컨테이너에서 안전하게 처리합니다. 이를 통해 웹의 편리함과 네이티브 프로그램의 강력함을 동시에 잡았습니다.

### [가치: 교육의 본질 회복]
이 플랫폼을 통해 선생님은 수업 준비 시간을 획기적으로 줄이고, 학생은 언제 어디서나 끊김 없는 코딩 학습을 이어갈 수 있습니다. DY Web IDE는 단순한 도구를 넘어, 학생들이 기술적 장벽 없이 창의적인 **문제 해결(Problem Solving)**이라는 소프트웨어 교육의 본질에 집중할 수 있도록 돕는 가장 든든한 파트너가 될 것입니다.

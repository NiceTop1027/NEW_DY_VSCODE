# 덕영고등학교 Web IDE 프로젝트

## 프로젝트 개요

**덕영고 Web IDE**는 학생들이 별도의 개발 환경 설정 없이 웹 브라우저만으로 프로그래밍을 배우고 실습할 수 있는 온라인 통합 개발 환경입니다.

## 왜 이 프로젝트를 만들었는가?

### 1. 개발 환경 설정의 장벽 제거

학교에서 프로그래밍을 가르칠 때 가장 큰 문제는 **개발 환경 설정**입니다.

- 학교 컴퓨터실에 VS Code, Python, Node.js 등을 설치하려면 관리자 권한이 필요
- 매번 다른 컴퓨터를 사용하면 설정을 처음부터 다시 해야 함
- 학생들이 집에서 공부하려면 또 설치 과정을 거쳐야 함
- 운영체제(Windows/Mac/Linux)마다 설치 방법이 달라서 혼란

**해결책**: 웹 브라우저만 있으면 어디서든 바로 코딩 시작!

### 2. 실시간 학습 피드백

전통적인 코딩 교육의 문제점:
- 코드를 작성하고 → 저장하고 → 터미널에서 실행하고 → 결과 확인
- 이 과정이 초보자에게는 복잡하고 시간이 오래 걸림

**해결책**:
- 코드 작성 즉시 실행 결과 확인
- HTML/CSS 실시간 프리뷰
- 에러 메시지를 바로 확인하고 수정

### 3. 협업과 공유의 편의성

- GitHub 연동으로 코드 버전 관리 학습
- 프로젝트를 쉽게 공유하고 제출
- 팀 프로젝트 협업 가능

### 4. 접근성과 비용

- **무료**: 별도의 라이선스나 구독 비용 없음
- **설치 불필요**: 인터넷만 되면 바로 사용
- **크로스 플랫폼**: Windows, Mac, 태블릿, 심지어 스마트폰에서도 사용 가능

## 주요 기능

### 코드 편집기
- **Monaco Editor** (VS Code와 동일한 엔진)
- 구문 강조, 자동 완성, 코드 접기
- 다양한 테마 지원
- 키보드 단축키 (Ctrl+S 저장, Ctrl+P 빠른 열기 등)

### 지원 언어
- **웹 개발**: HTML, CSS, JavaScript
- **프로그래밍**: Python, C, C++, Java
- **데이터**: JSON, YAML, Markdown

### 파일 시스템
- 가상 파일 시스템으로 프로젝트 관리
- 로컬 폴더 업로드 지원
- 파일 생성, 수정, 삭제, 이름 변경

### GitHub 연동
- OAuth 로그인
- 레포지토리 클론
- 변경사항 푸시
- 파일 선택 푸시

### 코드 실행
- JavaScript (QuickJS 엔진)
- Python (Pyodide)
- C/C++ (WASM 컴파일러)
- 실시간 터미널 출력

### 추가 기능
- **AI 어시스턴트**: 코드 설명, 오류 해결 도움
- **실시간 프리뷰**: HTML/CSS 즉시 미리보기
- **테마 커스터마이징**: 원하는 색상으로 변경
- **Zen 모드**: 집중 코딩을 위한 전체 화면

## 교육적 활용

### 수업 시나리오

**1. 기초 프로그래밍 수업**
```
1. 학생들에게 웹사이트 URL 공유
2. 브라우저에서 바로 접속
3. 예제 코드 복사하여 실행
4. 결과 즉시 확인하며 학습
```

**2. 과제 제출**
```
1. GitHub 계정 연동
2. 코드 작성 및 테스트
3. 커밋 메시지와 함께 푸시
4. 선생님이 GitHub에서 코드 리뷰
```

**3. 팀 프로젝트**
```
1. 팀원들이 같은 레포지토리 클론
2. 각자 파트 코딩
3. 변경사항 푸시
4. 코드 병합 및 리뷰
```

### 학습 효과

- **즉각적인 피드백**: 코드 실행 결과를 바로 확인하여 학습 효율 향상
- **실제 도구 경험**: VS Code와 유사한 인터페이스로 실무 환경 체험
- **협업 능력**: Git/GitHub 사용법 자연스럽게 학습
- **자기주도 학습**: 집에서도 동일한 환경에서 복습 가능

## 기술 스택

### Frontend
- **Monaco Editor**: VS Code 코어 에디터
- **Xterm.js**: 터미널 에뮬레이터
- **Webpack**: 모듈 번들러

### Runtime
- **QuickJS**: JavaScript 샌드박스 실행
- **Pyodide**: Python WASM 런타임
- **Emscripten**: C/C++ WASM 컴파일

### Storage
- **IndexedDB**: 영구 파일 저장
- **LightningFS**: Git 파일 시스템
- **isomorphic-git**: 브라우저 Git 클라이언트

### API
- **GitHub API v3**: OAuth, 레포지토리 관리
- **OpenAI API**: AI 어시스턴트 (선택)

---

## 보안 고려사항

- **XSS 방지**: 사용자 입력 이스케이프 처리
- **토큰 보안**: sessionStorage 사용, 로깅 제한
- **코드 샌드박스**: 사용자 코드 격리 실행
- **메모리 관리**: 이벤트 리스너 자동 정리

## 결론

**덕영고 Web IDE**는 단순한 코드 에디터가 아닙니다.

학생들이 프로그래밍의 본질인 **"문제 해결"**에 집중할 수 있도록, 환경 설정이라는 불필요한 장벽을 제거한 **교육용 플랫폼**입니다.

어디서든, 어떤 기기에서든, 브라우저만 열면 바로 코딩을 시작할 수 있습니다.
